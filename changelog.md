# Neat FFmpeg Changelog

## Методика версионирования

BREAKING CHANGE  → MAJOR  
feat             → MINOR  
fix              → PATCH  

Версии пишутся в формате:

1.0.2, где:
- **1 — MAJOR** (существенные изменения, нарушающие обратную совместимость)
- **0 — MINOR** (добавлены новые функции без нарушения совместимости)
- **2 — PATCH** (багфиксы, мелкие улучшения, правки интерфейса)

<!--
### Added       # новое
### Changed     # изменения поведения
### Deprecated  # устаревшее
### Removed     # удалённое
### Fixed       # багфиксы
### Security    # безопасность
-->

## [1.1.0] - 2025-12-18
### Added
- Добавлена вкладка «Скачивание» (поддерживается YouTube)  
  Работает на основе библиотеки `yt-dlp`, которая использует `deno` для обхода защиты

### Fixed
- Исправлен способ вставки ссылки:
  - ранее вставка работала только через `Ctrl + V`
  - комбинация срабатывала лишь при английской раскладке
  - теперь язык системы не имеет значения — ссылку можно вставить кликом ЛКМ в области ввода
