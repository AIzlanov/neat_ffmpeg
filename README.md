# Neat FFmpeg — Инструкция

Для обрезки и кодирования в проекте используется - [FFmpeg](https://github.com/FFmpeg/FFmpeg).  
Для скачивания видео с YouTube - [библиотека `yt-dlp`](https://github.com/yt-dlp/yt-dlp).  

## 0. Установка
1. Разархивируйте папку в любое удобное место.  
2. Найдите в корневой папке файл `main.py` (или `neat_ffmpeg.exe`, если вы собрали билд).  
3. Правой кнопкой по файлу → **"Отправить"** → **"Рабочий стол (создать ярлык)"**.  
   *(В Windows 11 сначала нажмите "Показать дополнительные параметры" внизу списка).*  

## 1. Запуск
Просто запустите созданный ярлык на рабочем столе.   
Программа сама подхватит необходимые кодеки из внутренней папки.   

## 2. Использование (обрезка)
Выбираем начало и конец отрывка в формате `ЧЧ:ММ:СС`.  
*Пример:* видео длится 5 минут, нужен кусок с 25-й секунды до 4:10.  
- **Начало:** 00:00:25  
- **Конец:** 00:04:10  
*(обрезка нескольких фрагментов за раз не поддерживается).*  

## 3. Настройки конвертации
По умолчанию выставлены параметры для лучшего баланса качества и веса (до 50 МБ).  

### Кратко о параметрах:
* **Constant Rate Factor (CRF):**  
    * 16 — исходное качество.  
    * 18 — очень высокое.  
    * 20–23 — хороший баланс (рекомендуется).  
    * 27+ — заметные артефакты, минимальный вес.  
* **Preset (Скорость сжатия):** Чем медленнее (slow), тем меньше весит итоговый файл при том же качестве.  
* **FPS:** Снижение с 60 до 30 кадров уменьшает вес почти вдвое.  
* **Resolution (Разрешение):**  
    * `1600:-1` — горизонтальное видео (ширина 1600).  
    * `-1:1600` или `custom -> 900:1600` — для вертикальных видео.
